## Компонент Button

### Описание:

Компонент отображающий куски кода с функционалом копирования.

### Параметры:

- `className?: string;`
- `'data-testid'?: string;`
- `text: string;` - содержимое

### Сторибук:

- `Default`

### Тесты:

- Компонент рендерится
- При нажатии на кнопку копирования текст добавляется в буфер
