## Компонент Avatar

### Описание:

Компонент отображающий аватар пользователя.

### Параметры:

- `className?: string;`
- `src?: string;`
- `alt?: string;`
- `size?: number;`

### Сторибук:

- `Default` - Отображение по умолчанию
- `WithImage` - Отображение с переданным изображениием
- `CustomSize300` - Отображение с переданным размером

### Тесты:

- Нет
